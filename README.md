# MOLD

A CEP panel plugin for Adobe Premiere Pro that lets you create and apply reusable folder structure templates directly inside your project.

Install it through [FORGE](https://github.com/Oxy720/FORGE-plugins) — the CEP plugin manager for Premiere Pro.

---

## What it does

Setting up the same folder structure for every new project gets tedious fast. MOLD lets you define a folder template once and apply it to any Premiere Pro project in seconds — keeping your bins clean and consistent across every job.

---

## Features

- Create reusable folder structure templates
- Apply templates to any Premiere Pro project
- Manage multiple templates for different project types
- Premiere Pro panel integration via CEP

---

## Installation

**Via FORGE (recommended)**

1. Open the FORGE panel in Premiere Pro
2. Find MOLD in the plugin list
3. Click **install**
4. Restart Premiere Pro

**Manual**

1. Download or clone this repo
2. Copy the folder to `%APPDATA%/Adobe/CEP/extensions/mold/` (Windows)
3. Restart Premiere Pro
